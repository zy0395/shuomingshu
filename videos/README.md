# 📹 视频文件存放说明

## 📂 此文件夹用于存放视频教程文件

### 建议的视频文件命名规范：

- ✅ `教程1-服务器基础配置.mp4`
- ✅ `教程2-TXT脚本入门.mp4`
- ✅ `教程3-LUA脚本开发.mp4`
- ✅ `教程4-游戏功能配置.mp4`

### 视频要求：

1. **格式**：MP4 格式（H.264编码，兼容性最好）
2. **分辨率**：建议 1280x720 (720p) 或 1920x1080 (1080p)
3. **大小**：单个视频建议不超过 50MB
4. **音频**：建议使用 AAC 音频编码

### 使用方法：

将您的视频文件复制到此文件夹后，修改 `pages/home.html` 文件中的视频地址：

```html
<!-- 原始代码 -->
<li><a href="#" onclick="loadVideo('视频地址1', '教程1：服务器基础配置'); return false;">

<!-- 修改为 -->
<li><a href="#" onclick="loadVideo('../videos/教程1-服务器基础配置.mp4', '教程1：服务器基础配置'); return false;">
```

详细说明请查看：`视频教程使用说明.md`
