## 更新部署命令

修改文件后，执行以下命令更新网站：

```bash
# 添加所有修改
git add -A

# 提交修改（替换为你的提交说明）
git commit -m "更新说明"

# 推送到 GitHub（网站会自动更新）
git push github main

# 同时推送到 Gitee（可选）
git push origin main
```

## 一键更新脚本

可以在目录下运行 `更新网站.bat` 快速更新。

## 注意事项
- GitHub Pages 更新后需要1-2分钟生效
- 大文件（>50MB）会有警告，但不影响使用
- 打包输出目录的 EXE 文件较大，建议添加到 .gitignore
