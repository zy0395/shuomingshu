# 应用图标说明

## 📌 需要准备应用图标

在打包之前，请将您的应用图标文件命名为 `icon.ico` 并放置在此目录下。

### 图标要求

- **格式**：ICO 格式
- **推荐尺寸**：256x256 像素
- **文件名**：icon.ico

### 如何制作图标

#### 方法1：在线转换（推荐）

1. 准备一张 PNG 或 JPG 图片（建议 256x256 或 512x512）
2. 访问在线转换工具：
   - https://convertio.co/zh/png-ico/
   - https://www.aconvert.com/cn/icon/png-to-ico/
   - https://www.favicon-generator.org/
3. 上传图片并转换为 ICO 格式
4. 下载转换后的文件
5. 重命名为 `icon.ico` 并放在此目录

#### 方法2：使用 Photoshop

1. 在 PS 中创建 256x256 的画布
2. 设计您的图标
3. 安装 ICO 插件
4. 另存为 ICO 格式

#### 方法3：使用免费工具

1. 下载 IcoFX（免费版）：https://icofx.ro/
2. 创建或导入图片
3. 导出为 ICO 格式

### 临时解决方案

如果暂时没有图标，可以：
1. 使用 Windows 自带的图标（从系统文件中复制）
2. 从网上下载免费图标（注意版权）
3. 使用默认的 Electron 图标（不推荐用于发布）

### 多尺寸图标

为了在不同场景下显示最佳效果，ICO 文件应包含多个尺寸：
- 16x16
- 32x32
- 48x48
- 64x64
- 128x128
- 256x256

大多数在线工具会自动生成多尺寸图标。
